# 🛒 Ecommerce App

A modern and responsive Ecommerce web application built using React and Vite.

## 🚀 Features

- 🏠 Home Page
- 🛍️ Product Listing & Details
- 🛒 Cart Management
- ❤️ Wishlist with Local Storage
- 🔍 Search and Filtering
- 📦 Order History
- 👤 Profile & Settings
- 🔐 Route Protection
- 📊 Admin Dashboard & Analytics
- 🔔 Toast Notifications
- 📱 Responsive Design
- ♿ Accessibility Support

---

## 🛠️ Tech Stack

- React.js
- Vite
- React Router DOM
- Context API
- React Toastify
- React Window
- Chart.js
- CSS3
